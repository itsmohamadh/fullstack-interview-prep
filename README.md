# Fullstack Interview Prep

> The 20% that shows up 80% of the time.

This repository is a **roadmap**, **cheatsheet**, and **last-minute reminder** for fullstack interviews. Each topic includes just enough to trigger your memory. If something is unfamiliar, search the web and dive deeper. [Philosophy →](philosophy.md)

## 📖 How to use this repo

### New to interview prep?

Start here → [`frontend/js/questions.md`](frontend/js/questions.md)

### Have an interview in 1-2 days?

Skip the coding folder. Read the `questions.md` files under each topic as a final refresher.

### Have 1-2 weeks?

Work through the topics in this order:

1. [JavaScript](frontend/js/questions.md) + [Coding challenges](frontend/js/coding/)
2. [React](frontend/react/questions.md)
3. [TypeScript](frontend/ts/questions.md)
4. [CSS](frontend/css/questions.md)
5. [Web](frontend/web/questions.md)
